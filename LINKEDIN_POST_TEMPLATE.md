## Post Draft
I just completed **Task 2: Customer Retention & Churn Analysis** for the Data Science & Analytics track by Future Interns.

In this project, I analyzed subscription-style customer data to answer key business questions:
- Why customers churn
- Which segments are highest risk
- What improves retention and customer lifetime value

### What I built
- End-to-end Python analytics pipeline
- Interactive retention dashboard
- Churn and cohort-proxy analysis report
- Actionable strategy recommendations for reducing customer loss

### Key insights
- Overall churn: **26.54%**
- Highest churn concentration: **first 3 months of tenure**
- Strong churn drivers: **month-to-month contracts**, **electronic check payments**, and missing **security/support add-ons**

### Tools used
Python, Pandas, NumPy, Seaborn, Matplotlib, Plotly, GitHub Actions, GitHub Pages
